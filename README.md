# php-socket-anon-chat
for example

```bash
git clone https://github.com/aethletic/php-socket-anon-chat.git chat
```

```bash
cd chat && composer install
```

```bash
php app.php start
```

```bash
cd public && php -S localhost:1337
```

Открыть в браузере http://localhost:1337/
